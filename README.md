# RewardSphere

Mobile + Web rewards & cashback demo built with **Flutter** (single codebase) and **Node.js / Express** backend using **Postgres (Supabase)**.

This repo contains:
- `client/` — Flutter app (Android / iOS / Web)
- `server/` — Node.js API (Express + pg)
- `.github/workflows/` — CI templates

## Quick start (local)

### Prereqs
- Git
- Flutter (stable) — run `flutter doctor`
- Node.js (LTS) + npm
- Optional: Docker (for running Postgres locally)

### 1. Clone
```bash
git clone https://github.com/shaikhsadakat/rewardsphere.git
